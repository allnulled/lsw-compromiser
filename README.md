# lsw-compromiser

Promise manager tool for LSW.

## Installation

```sh
npm i -s @allnulled/lsw-compromiser
```

## Importation

From node.js:

```js
require("@allnulled/lsw-compromiser");
```

From html:

```js
<script src="node_modules/@allnulled/lsw-compromiser/lsw-compromiser.js"></script>
```
